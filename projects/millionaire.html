Millionaire in C

First programming project. Developed in the Linux terminal using the C programming language. The game "Millionaire in C" is a quiz game where the player must answer a series of questions with increasing difficulty to win a prize. The game has a total of 15 questions, and the player can use three lifelines to help them answer the questions: 50/50, phone a friend, and ask the audience. The game is designed to be fun and educational, as it helps players improve their general knowledge and logical reasoning skills.

This game was taken to the research area where students from the A-2018 semester used it, generating results that show that the game Millionaire in C positively impacts logical reasoning.
